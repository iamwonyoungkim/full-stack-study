\' : Single quote
\" : Double quote
\\ : Backslash
\n : New line
\t : Tab

\r : Carriage return
\b : Backspace
\f : Form feed
\v : Vertical tab (rarely used)
\0 : Null character (not the same as an empty string)
\uXXXX : Unicode character, where XXXX is a 4-digit hexadecimal code
\xXX : Latin-1 character, where XX is a 2-digit hexadecimal code
